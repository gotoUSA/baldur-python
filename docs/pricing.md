## How it works

Baldur is a Python package — not a hosted service. Install it with `pip`; PRO is
unlocked with a license key delivered after purchase, and the framework runs
entirely inside your own application. Payments, invoicing, and tax are handled by
**Creem**, our Merchant of Record.
